# Plan Template

Use this template to create structured plans for features or projects. Replace placeholder content with actual details.

## Goals

Describe the overarching goals and desired outcomes of the work. Link these goals back to the value proposition and user personas.

## Requirements

List functional and non‑functional requirements. Include constraints and assumptions.

## Milestones

Break the work down into milestones with clear deliverables and approximate timelines. Use checkboxes to track progress.

- [ ] Milestone 1 – description and target date
- [ ] Milestone 2 – description and target date

## Risks and Mitigations

Identify potential risks, their likelihood and impact, and describe mitigations.

## Tools and Resources

List any tools, libraries, skills or personnel required to complete the work.

## Acceptance Criteria

Define the conditions that must be met for the work to be considered complete. Align these criteria with the goals and requirements.
